## 项目介绍
基于网易云api开发，模仿网易云pc端。<br>
运行该项目前请下载[网易云api](https://github.com/Binaryify/NeteaseCloudMusicApi)
## 安装

```
git clone 仓库地址<br>
npm install
```
## 运行
```
npm run serve
```
